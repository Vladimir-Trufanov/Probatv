## Как в html подгрузить другой html

Подгрузить другой HTML-документ в HTML можно с помощью разных методов: 

- тег:

```
 <link rel="import">
 ``` 
- JavaScript (метод fetch())

- тег:
 
```
<iframe>
```
Выбор метода зависит от задачи: для интеграции без перезагрузки страницы подходит JavaScript, для изолированного отображения содержимого - iframe. 

1. Тег "import" позволяет подключить любой документ к странице. Синтаксис: 

```
<link rel="import" href="http://site.com/document.html">
```
Особенности:

HTML, загруженный через "import", имеет отдельный DOM-документ, но скрипты в нём выполняются в общем контексте страницы.

Файл обрабатывается, выполняются скрипты, строится DOM-документ, но не показывается, а записывается в свойство link.import.

Стили импорта попадают в контекст страницы.

Повторный импорт с тем же URL использует уже существующий документ, без повторного выполнения скриптов в нём.
 
Важно: тег "import" может быть в любом месте документа, даже в ***head***. 

2. С помощью JavaScript (fetch). Метод fetch() позволяет асинхронно получать ресурсы по сети, в том числе HTML. Чтобы загрузить HTML, нужно: 
- Вызвать fetch() с путём к ресурсу.
- Использовать метод response.text() - он преобразует ответ в строку с HTML.
- Вставить HTML в DOM с помощью, например, свойства Element.innerHTML.
 
Пример: в HTML-файле есть элемент с идентификатором id_content, и в скрипте используется fetch('someFile.html') — после загрузки HTML-кода содержимое элемента заменяется. 

3. С помощью iframe. Тег ***iframe*** создаёт контейнер, внутри которого можно разместить другую веб-страницу. Некоторые особенности: 

- Атрибут src определяет URL страницы, которая будет встроена.
- Атрибут srcdoc позволяет встроить HTML-код целиком, так что браузеру не придётся подгружать что-то по внешней ссылке.
- Атрибут sandbox устанавливает ограничения для содержимого в iframe, предотвращая его взаимодействие с основной страницей.
- Атрибут loading оптимизирует загрузку контента, например, loading=“lazy” задерживает её до момента, когда элемент станет видимым на экране пользователя.
 
Важно: iframe - это как будто ещё одна страница, которая загружается внутри основной страницы, поэтому содержимое фрейма может загружаться дольше, чем сайт.
