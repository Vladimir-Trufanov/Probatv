## Утечки памяти

Фрагмент кода по состоянию памяти:

```
   // 2026-02-11
   $p1=memory_get_usage(false);
   prown\ConsoleLog('Память, занимаемая кодом PHP: '.$p1);
   $p2=memory_get_usage(true);
   prown\ConsoleLog('Память, выделенная для сайта: '.$p2);
   $p3=memory_get_peak_usage(false);
   prown\ConsoleLog('Пик потребления памяти:       '.$p3);
   $p4=memory_get_peak_usage(true);
   prown\ConsoleLog('Пик выделения памяти сайту:   '.$p4);
```

Ответ на моноблоке:  (выделяется 2 Мб)

```
SignaPhoto.php:37 Память, занимаемая кодом PHP: 1116680
SignaPhoto.php:41 Память, выделенная для сайта: 2097152
SignaPhoto.php:45 Пик потребления памяти:       1190240
SignaPhoto.php:49 Пик выделения памяти сайту:   2097152
```
